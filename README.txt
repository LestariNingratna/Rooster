Thank you for downloading my game, "The Adventure of a Rooster"
This is a command line interactive game written in Python.

To get started:
1. Unzip/Extract the files within this folder to your computer
2. If you don't have Python 2 installed, please install it from http://www.python.org/download
3. Open the file named "rooster.py"
4. You can also open it with command line ([your_directory](enter)cd [your folder](enter)rooster.py
   for example E:(press enter)rooster(press enter)rooster.py(press enter)
5. Have fun!
